# Whats - WhatsApp Desktop

Unofficial webapp for WhatsApp Web.
<br><br>

### Required packages:

- python >= 3.6
- PyQt5 >= 5.15.5
- PyQtWebEngine >= 5.15.5
<br><br>

### Installation instructions:

To install "Whats", use the following commands:
```sh
$ git clone https://github.com/mxnt10/whats.git
$ cd whats

$ sudo ./install.sh
```
